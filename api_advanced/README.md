# API Advanced

This project contains Python scripts utilizing the Reddit API to interact with subreddits, parse JSON data, implement recursive API calls, handle pagination, and sort dictionaries by value.

## Requirements
* Allowed editors: `vi`, `vim`, `emacs`
* Interpreted on Ubuntu 14.04 LTS using `python3` (version 3.4.3)
* All files must end with a new line
* The first line of all files must be exactly `#!/usr/bin/python3`
* Libraries imported must be organized in alphabetical order
* Code must use the **PEP 8** style
* All files must be executable
